TA Timer

Instructions

1. Please select your preferred mode.

    Stopwatch: Counts up to a given point in time (24-hr).
                          e.g. 5 pm = 17:00:00

    Countdown: Counts down from the given time amount.

2. Enter the endtime / countdown amount.

3. Type anything you want inside the text box below.
    The size can be adjusted by dragging on the bottom right corner.
    (Feature not supported on Internet Explorer)

4. You can adjust the text size by pressing the + and - buttons.
